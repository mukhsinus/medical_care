## 🎯 НУЖНЫЕ ДЕЙСТВИЯ (3 шага)

### 1️⃣ Получить ИКПУ код
```
Перейти: https://merchant.paycom.uz
Создать аккаунт/войти
Найти: Merchant ID (16 цифр)
Пример: 507144111111111
```

### 2️⃣ Обновить .env
```env
PAYME_MERCHANT_ID=ваш_16_значный_код
PAYME_KEY=ваш_api_ключ
PAYME_TEST_MODE=false
```

### 3️⃣ Установить Callback URL
```
В Payme cabinet:
https://your-domain.com/api/payments/payme/callback
```
